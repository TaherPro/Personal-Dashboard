# Personal Smart Dashboard

A lightweight web app built with **TypeScript** and **Vanilla JavaScript (ES6+)**.  
This project demonstrates modern frontend development concepts.



## Features

**Notes Manager**
  - Add and delete notes
  - Persistent storage using `localStorage`

**Todo List**
  - Add, complete, delete tasks
  - Filter by All / Active / Completed

**Bookmarks Manager**
  - Save bookmarks with title, URL, and category
  - Prevent duplicate URLs using `Set`
  - Category summary using `Map`

**Random Quotes**
  - Fetch random quotes from a public API
  - Local fallback quotes if API fails

**Weather Widget**
  - Search weather by city name
  - Uses Open-Meteo API (no API key required)
  - Displays temperature, wind speed, and condition

**Light / Dark Theme Switcher**
  - Implemented with CSS variables
  - Theme persisted in `localStorage`
  - Built using TypeScript `enum`
